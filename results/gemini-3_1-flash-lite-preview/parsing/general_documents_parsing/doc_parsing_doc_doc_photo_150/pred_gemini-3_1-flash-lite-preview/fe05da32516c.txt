**Problem 1**

In the lecture "syntax" I develop a toy language Little English. It has the following lexicon (each expression belongs to exactly one grammatical category: names, predicates, or connectives).

Names: **Kate**, **Billy**
Predicates: **smokes**, **sleeps**
Connectives: **and**, **or**

Furthermore I define well-formed formula in Little English as follows:
1. If $\Phi$ is a name, and $\Psi$ is a predicate, then "$\Phi \Psi$" is a well-formed formula (wff).
2. If $\Phi$ and $\Psi$ are well-formed formulas, and $\Omega$ is a connective, then $\Phi \Omega \Psi$ is a wff.
3. Nothing else is a wff.

I pointed out that the expression **Kate sleeps and smokes** is not a wff.

This is counterintuitive, **Kate sleeps and smokes** is a perfectly grammatical sentence of English. Fortunately, there is an easy way to fix this. We can add the following condition to the definition of wff.

(C) If $\Phi$ is a well-formed formula, $\Psi$ is a predicate, and $\Omega$ is a connective, then $\Phi \Omega \Psi$ is a wff.

Which yields the following revised definition of wff:

1. If $\Phi$ is a name, and $\Psi$ is a predicate, then "$\Phi \Psi$" is a well-formed formula (wff).
2. If $\Phi$ and $\Psi$ are well-formed formulas, and $\Omega$ is a connective, then $\Phi \Omega \Psi$ is a wff.
3. If $\Phi$ is a well-formed formula, $\Psi$ is a predicate, and $\Omega$ is a connective, then $\Phi \Omega \Psi$ is a wff.
4. Nothing else is a wff.

a) Using the revised definition, show that **Kate sleeps and smokes** is a wff.
b) Using the revised definition, show that **Kate smokes or Bill sleeps and smokes** is a wff.
c) Explain why **Bill or Kate sleeps** is not a wff under the revised definition.
d) Further revise the revised definition, so that **Bill or Kate sleeps** counts as a wff. You have to add to it a new condition, like I did with (C) above.
e) Using your new definition, show that **Bill or Kate sleeps** is a wff.
