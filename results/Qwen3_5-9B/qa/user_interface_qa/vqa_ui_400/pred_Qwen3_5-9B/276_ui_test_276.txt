The **Home test** is the antibody test that involves finger prick testing.

As shown in the infographic:

- Under the **“Antibody test”** column → **“Home test”** sub-column:
  - **How does the test work?** → *“‘Finger-prick’ test taken at home”*
  - **Can I get tested now?** → Currently not accurate enough, but planned for future use.
  - **What does it mean?** → Tells if you’ve had the virus (not immunity level); helps scientists understand spread.

So, the **Home antibody test** is the one that uses a finger prick.
