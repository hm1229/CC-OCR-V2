First, the infographic states **images with faces on Instagram get 38% more likes than images without faces**.
If an image without faces gets 100 likes, an image with faces gets 138 likes. The inverse percentage (what percentage of the with-faces like count a no-face image gets) is:
```
100 / 138 * 100 ≈ 72.5%
```
If you mean the inverse of the 38% uplift (the percentage decrease in likes for images without faces, relative to images with faces), that is ~27.5%.
The most direct answer to the inverse percentage of the 38% extra gain is ~72% (or ~72.5% for precision).
