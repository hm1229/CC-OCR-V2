{\large \textbf{1. NUMBER SYSTEM}}
\end{center}

\begin{center}
\textsc{``Number Rules The Universe.'' Pythagoras}
\end{center}

\textbf{Natural Numbers}

\quad These are also called \textit{Counting numbers} 1, 2, 3, 4, 5, \ldots and are known as natural numbers. The set of all natural numbers can be represented by N.
\begin{center}
$\text{N} = \{1, 2, 3, 4, 5, \ldots\}$
\end{center}

\textbf{Whole Numbers}

\quad If we include 0 among the natural numbers, then the numbers 0, 1, 2, 3, 4, 5, \ldots are called whole numbers. The set of whole numbers can be represented by w. Clearly, every natural number is a whole number but 0 is a whole number which is not a natural number.
\begin{center}
$\text{W} = \{0, 1, 2, 3, 4, 5, \ldots\}$
\end{center}

\textbf{Integers}

\quad All counting numbers and their negatives including zero are known as integers. Z or I. The set of integers can be represented by
\begin{center}
$\text{Z or I} = \{\ldots -4, -3, -2, -1, 0, 1, 2, 3, 4, \ldots\}$
\end{center}
\begin{itemize}
\item \textbf{Positive Integers:} The set $\text{I}+ = \{1, 2, 3, 4, \ldots.\}$ is the set of all positive integers.
\item \textbf{Negative Integers:} The set $\text{I}- = \{-1, -2, -3, \ldots\}$ is the set of all negative integers. 0 is neither positive nor negative.
\item \textbf{Non-negative Integers:} The set $\{0, 1, 2, 3, \ldots\}$ is the set of all non-negative integers.
\end{itemize}

\textbf{Note:} Only whole numbers are integers. Therefore,: fractions, decimals, percent and exponents can never be integers.

\textbf{Rational Numbers}

\quad The numbers of the form $p/q$, where p and q are integers and $q \neq 0$, are known as rational numbers, e.g. $\dfrac{4}{7}, \dfrac{3}{2}, \dfrac{5}{8}, \dfrac{0}{1}, \dfrac{2}{3},$ etc. The set of all rational numbers is denoted by Q.
