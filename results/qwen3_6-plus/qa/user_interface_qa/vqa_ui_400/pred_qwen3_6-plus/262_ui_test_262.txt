According to the infographic, there are **15,000** WiFi-equipped public libraries in the U.S.

This is shown in the bottom section under “In some communities...”, where it compares:

- McDonald’s: 12,000 WiFi locations  
- Starbucks: 7,000 WiFi locations  
- **Public libraries: 15,000 WiFi locations**

The text also notes that McDonald’s and Starbucks *combined* have more WiFi locations than all public libraries — which mathematically checks out (12,000 + 7,000 = 19,000 > 15,000).

✅ So, the answer is: **15,000**
