Actually, the infographic doesn’t describe “three basic approaches for organizing a **network**.” Instead, it explains the **three basic steps all modern search engines use to organize websites** — which is about how search engines process and structure web content, not how computer networks are organized.

Those three steps are:

1. **Crawling**  
   → Search engines follow links to browse the 60 trillion (and growing) pages that make up the web.

2. **Sorting**  
   → Results get sorted by a variety of factors (like relevance, keywords, backlinks, etc.).

3. **Indexing**  
   → Sorted pages get stored in an index so search engines can match queries with results faster.
