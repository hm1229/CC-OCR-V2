### 儿童锁  
您的电器均配备儿童锁，使儿童无法意外开启电器或更改任何设置。  

#### 提示  
- 您可以在基本设置中设定是否通过该选项设置儿童锁。→ 第 17 页的“基本设置”  
- 如果连接了电磁灶，电磁灶不会受烤箱上的儿童锁的影响。  
- 切断电源后，儿童锁便告失效。  


#### 启用和停用  
如要激活儿童锁，必须将功能选择按钮转到“关”位置。按下并按住 $\boldsymbol{\text{⚙}}$ 按键约 4 秒。代表该功能的符号将出现在显示屏上。儿童锁现已启用。  

**提示**：如果已经设置定时器持续时间 $\boldsymbol{\text{⏰}}$，该时间将继续倒计时。启用儿童锁后，无法再更改定时器持续时间。  

如要取消激活儿童锁，再次按下并按住 $\boldsymbol{\text{⚙}}$ 按键 4 秒，直到该符号在显示屏上熄灭。  


### 基本设置  
为了帮助您有效、简单地使用电器，我们为您提供了各种设置。您可以根据需要更改这些设置。根据电器的功能，并非所有基本设置都可供访问。  


#### 基本设置列表  
| 基本设置 | 选项 |  
| --- | --- |  
| $\boldsymbol{c1}$ 显示时间 | $0 = \text{隐藏时间}$<br>$1 = \text{显示时间}^*$ |  
| $\boldsymbol{c2}$ 烹饪时间或定时器时间到后的信号音持续时间 | $1 = \text{约 10 秒钟}$<br>$2 = \text{约 30 秒钟}^*$<br>$3 = \text{约 2 分钟}$ |  
| $\boldsymbol{c3}$ 触碰按键时的按键音 | $0 = \text{关}$<br>$1 = \text{开}^*$ |  
| $\boldsymbol{c4}$ 冷却风扇持续运行时间 | $1 = \text{短}$<br>$2 = \text{中}$<br>$3 = \text{长}^*$<br>$4 = \text{超长}$ |  
| $\boldsymbol{c5}$ 应用某个设置之前需要等待的时间 | $1 = \text{约 3 秒钟}^*$<br>$2 = \text{约 6 秒钟}$<br>$3 = \text{约 10 秒钟}$ |  
| $\boldsymbol{c6}$ 激活儿童锁 | $0 = \text{否}$<br>$1 = \text{是}^*$ |  
| $\boldsymbol{c7}$ 水硬度 $^{**}$ | $0 = \text{软化}$<br>$1 = \text{软（最高 1.3 mmol/l）}$<br>$2 = \text{中（1.3 - 2.5 mmol/l）}$<br>$3 = \text{硬（2.5 - 3.8 mmol/l）}$<br>$4 = \text{非常硬}^*（高于 3.8 mmol/l）$ |  


**说明**：  
- $^*$ 出厂设置（出厂设置可能根据电器型号的不同而不同）  
- $^{**}$ 并非所有电器型号可用。  


#### 更改基本设置  
必须将功能选择按钮转到“关”位置。  

1. 按下并按住 $\boldsymbol{\text{⚙}}$ 按键大约 4 秒钟。将第一个基本设置显示在显示屏中，如 $c1!$。  
2. 使用 $-$ 或 $+$ 按键更改基本设置。  
3. 按下 $\boldsymbol{\text{⚙}}$ 按键确认。  
4. 使用 $\boldsymbol{\text{⚙}}$ 按键继续按照说明完成所有基本设置，必要时使用 $-$ 或 $+$ 按键更改设置。  
5. 结束后，再次按下并按住 $\boldsymbol{\text{⚙}}$ 按键约 4 秒，确认设置。所有基本设置现已生效。您可以随时再次更改基本设置。  

**提示**：断电后，将恢复基本出厂设置。
