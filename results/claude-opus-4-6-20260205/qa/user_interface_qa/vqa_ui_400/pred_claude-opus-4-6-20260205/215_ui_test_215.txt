# Three Basic Approaches for Organizing Websites

According to the infographic, all modern search engines use the same basic approach to organizing websites, which consists of three steps:

1. **Crawling** - Search engines follow links to browse the 60 trillion (and growing) pages that make up the web.

2. **Sorting** - Results get sorted by a variety of factors (more on this in a second).

3. **Indexing** - Sorted pages get stored in an index so search engines can match queries with results faster.
