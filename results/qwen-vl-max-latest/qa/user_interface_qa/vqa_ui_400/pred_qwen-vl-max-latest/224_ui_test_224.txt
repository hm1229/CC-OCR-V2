The names of the search technologies before 1995 mentioned in the infographic are:

1. **Archie** (pre-WWW) - The first search engine, which didn't crawl sites but kept an index of public FTP servers.
2. **Lycos** - The Lycos index grew from 54,000 documents in 1994 to 60 million by 1995.

These were the primary search technologies highlighted prior to 1995.
