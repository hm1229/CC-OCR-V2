Before 1995, there were two significant search engine technologies:

1. **Archie (Pre-WWW)**: Developed in 1990, it was the first search engine that indexed FTP sites, allowing users to find and download files.

2. **Yahoo!**: Launched in 1994, it initially focused on curating a directory of websites rather than indexing them, but it played a crucial role in the early organization of the internet.

These technologies laid the groundwork for the more advanced search engines that emerged later.
