1) The range of the function $\dfrac{1}{1-2\sin x}$ is

\item[d)] $(-\infty,-1]\cup\left[\dfrac{1}{3},\infty\right)$
\end{itemize}

2) The function $f : [0,2\pi] \to [-1,1]$ defined by $f(x)=\sin x$ is
\begin{itemize}
\item[a)] one-to-one
\item[b)] onto
\item[c)] bijection
\item[d)] cannot be defined
\end{itemize}

\noindent
3) The solution set of the following inequality $|x-1| \geq |x-3|$ is
\begin{itemize}
\item[a)] $[0,2]$
\item[b)] $[2,\infty)$
\item[c)] $(0,2)$
\item[d)] $(-\infty,2)$
\end{itemize}

\noindent
4) If $3$ is the logarithm of $343$, then the base is
\begin{itemize}
\item[a)] $5$
\item[b)] $7$
\item[c)] $6$
\item[d)] $9$
\end{itemize}

\noindent
5) In a $\triangle ABC$, $\tan\left(\dfrac{A}{2}\right)=$
\begin{itemize}
\item[a)] $\sqrt{\dfrac{(s-b)(s-c)}{bc}}$
\item[b)] $\sqrt{\dfrac{s(s-a)}{bc}}$
\item[c)] $\sqrt{\dfrac{(s-b)(s-c)}{s(s-a)}}$
\item[d)] $\sqrt{s(s-a)(s-b)(s-c)}$
\end{itemize}

\noindent

\item[b)] $40$
\item[c)] $39$
\item[d)] $38$
\end{itemize}

\item[d)] $\dfrac{n^2-n+2}{2}$

\item[c)] $1,\dfrac{1}{2}$
