# Route Shapes in Long Routes

Based on the infographic about the Best Cycling Routes in Ireland, the **Long Routes** have the following route shapes:

1. **Ring of the Reeks Cycle**, MacGillycuddy Reeks, Co. Kerry - **LOOP**
2. **Burren Cycling Route, Long Loop**, Co. Clare - **LOOP**
3. **Inishowen Scenic Cycle**, Co. Donegal - **LOOP**
4. **Ring of Kerry Scenic Cycle**, Co. Kerry - **LOOP**

All four long routes have a **LOOP** route shape.
