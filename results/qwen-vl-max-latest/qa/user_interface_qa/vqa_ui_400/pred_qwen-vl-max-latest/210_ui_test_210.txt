The infographic mentions **three** basic approaches for organizing websites used by modern search engines:

1. **Crawling** – Search engines follow links to browse the web and gather information from pages.
2. **Sorting** – Results are sorted based on various factors.
3. **Indexing** – Sorted pages are stored in an index so that search engines can match queries with relevant results.

These three steps are highlighted under the section titled "SEARCH ENGINE BASICS."
