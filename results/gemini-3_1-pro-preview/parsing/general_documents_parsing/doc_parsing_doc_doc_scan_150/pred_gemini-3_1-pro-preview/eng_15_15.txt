    \begin{itemize}
        \item Flash and Indicator 2in1 application
        \item Dual Independent 1.0A LED1/LED2 Current Source
        \begin{itemize}
            \item Flash: 3.91mA$\sim$1.0A, 256 levels 3.91mA/level
            \item Torch: 1.46mA$\sim$375mA, 256 levels 1.46mA/level
            \item Indicator: 0.37mA$\sim$375mA, 1024 levels 0.37mA/level
            \item Flash Timeout: 40ms$\sim$1.6s, 16 levels
        \end{itemize}
        \item Independent 375 mA LED3 Current Sink
        \begin{itemize}
            \item Flash: 1.46mA$\sim$375mA, 256 levels 1.46mA/level
            \item Torch: 1.46mA$\sim$375mA, 256 levels 1.46mA/level
            \item Indicator: 0.73mA$\sim$375mA, 512 levels 0.73mA/level
            \item Flash Timeout: 40ms$\sim$1.6s, 16 levels
        \end{itemize}
        \item Optimized Flash LED Current During Low Battery Conditions (IVFM)
        \item Hardware Strobe Enable (STROBE)
        \item 400kHz I$^2$C: AW36526 (I$^2$C Address=0x63)
        \item 0.4mm Pitch, FCQFN-10L Package
    \end{itemize}

\begin{minipage}[t]{0.48\textwidth}
    \textcolor{blue}{\textbf{\large GENERAL DESCRIPTION}}
    \vspace{0.2cm}
    
    The AW36526 is a inductorless triple channel flash \& Indicator 2in1 LED driver that provides a high level of adjustability within a ultra-small solution size. The AW36526 includes two current sources and a current sink which is controlled independently by I$^2$C-compatible interface. The AW36526 provides IVFM protection mode to prevent system reset or shutdown under low battery condition.
    
    \vspace{0.2cm}
    The AW36526 is controlled via an I$^2$C-compatible interface. The main features of the AW36526 include: flash / torch / IR / Indicator current, flash timeout duration, IVFM and breath features. The AW36526 also provides hardware flash pin (STROBE) to control flash event.
    
    \vspace{0.2cm}
    The device operates over a $-40^\circ$C to $+85^\circ$C ambient temperature range.
    
    \vspace{0.2cm}
    The AW36526 is available in small 0.4mm pitch 1.6mm$\times$1.2mm FCQFN-10L package.
\end{minipage}
