# CC-OCR V2 Datasets

The full datasets (including images and annotations) are hosted on Hugging Face. You can download them from the following link:

[![Hugging Face](https://img.shields.io/badge/Hugging%20Face-Dataset-yellow)](https://huggingface.co/datasets/Eioss/CC-OCR-V2)

Please download and extract the dataset into this directory. After extraction, the directory structure should look like this:

```text
ocr_datasets/
├── README.md
├── extraction/
├── grounding/
├── parsing/
├── qa/
└── recognition/
```
