\section*{八、无牙区牙槽嵴的楔形手术}

\subsection*{邻近基牙的无牙区}
为了保持牙弓无牙区的健康牙周组织，邻近基牙的无牙区牙槽嵴形态是应予考虑的重要方面$^{28}$，楔形手术是消除无牙区牙周袋的一种方法，它也用于邻近上颌结节及磨牙后三角区基牙的牙周组织外形修整。

邻近基牙的无牙区伴随着牙周治疗后复发的牙周病往往形成深的牙周袋。

\subsection*{邻近基牙无牙区存在的问题}
\begin{enumerate}[label=(\arabic*)]
\item 菌斑难以控制。
\item 因为进行刮治和根面平整时器械的通路有限，因此基础治疗的效果可能并非最佳。
\item 上颌结节和磨牙后三角区覆盖着厚的牙龈，往往形成深牙周袋，因此，经常有进行性的根分叉病变。
\item 缺牙区附近的基牙是咬合的关键牙并且承担应力，所以有进展成重度牙周病的高风险。
\end{enumerate}

因此，为了保持邻近无牙区牙槽嵴的基牙周围有良好的牙周环境，推荐用楔形手术消除该区的牙周袋。

\subsection*{楔形手术的目的}
\begin{enumerate}[label=(\arabic*)]
\item 消除牙周袋。
\item 维护和保存附着龈。
\item 便于器械进入。
\item 延长临床牙冠。
\item 创建易于清洁的牙龈－牙槽骨形态。
\end{enumerate}

该手术可以去除邻近基牙无牙区的厚牙龈组织。如有骨缺损，也可通过平整来纠正骨外形、消除骨内缺损。牙周袋消除后，形成浅的龈沟，利于术后维护。

为了初期愈合，薄组织瓣能与牙根和骨面有更好的贴附：这不仅能保证消除牙周袋，还能减轻疼痛，缩短愈合时间。

\subsection*{楔形手术组织瓣的设计}
楔形手术组织瓣有四种设计方式：方形、线形、三角形及带蒂瓣（表1-12）。组织瓣的设计取决于无牙区牙槽嵴的大小、上颌结节、磨牙后三角区以及软组织的厚度。

\subsection*{楔形手术中决定组织瓣设计的因素}
\begin{enumerate}[label=(\arabic*)]
\item 大小与形态。
\item 软组织厚度。
\item 通路进入的难度。
\item 基牙的附着龈宽度。
\item 在基牙的无牙区侧牙周袋深度和骨缺损程度。
\item 作为修复治疗的基牙需要的临床牙冠长度。
\end{enumerate}

由于实用，作者经常应用方形切口（表1-12）。

\subsection*{方形切口的适应证}
\begin{enumerate}[label=(\arabic*)]
\item 长和大的无牙区牙槽嵴、上颌结节、磨牙后三角区。
\item 在楔形区须去除较多组织。
\item 附着龈足够宽。
\item 在基牙的近中和远中存在深的牙周袋和骨缺损。
\end{enumerate}

方形切口由两个平行的内斜切口和一个垂直切口组成，垂直切口是一个松弛切口以帮助组织瓣更贴合于术区。被去除的楔形组织量（两个内斜切口之间的范围）的多少取决于多种因素。
