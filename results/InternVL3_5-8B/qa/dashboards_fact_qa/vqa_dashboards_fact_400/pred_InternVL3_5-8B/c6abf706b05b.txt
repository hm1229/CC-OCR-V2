Before 1995, the notable search technologies include:

1. **Archie (Pre-WWW)**: Launched in 1990, it was the first search engine that indexed FTP sites, allowing users to find and download files.

2. **WorldWideWeb**: Although not a search engine itself, it was an early web browser that laid the groundwork for the web's development.

These early technologies set the foundation for the more advanced search engines that emerged later.
