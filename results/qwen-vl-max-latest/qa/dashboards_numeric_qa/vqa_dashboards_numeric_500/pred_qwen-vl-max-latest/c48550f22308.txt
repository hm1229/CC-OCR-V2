The minimum number of host processors that can be connected in **Microsoft Hyper-V** is **4**, as shown in the comparison table under the "Max host processors" feature.

This applies to all Microsoft Hyper-V editions listed:
- **Microsoft Hyper-V Standard**
- **Microsoft Hyper-V Enterprise**
- **Microsoft Hyper-V DC**

So, the **minimum** number of host processors supported by Microsoft Hyper-V is **4**.
