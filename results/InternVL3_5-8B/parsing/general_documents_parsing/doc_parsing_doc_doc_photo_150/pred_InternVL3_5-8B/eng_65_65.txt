    If we include 0 among the natural numbers, then the numbers 0, 1, 2, 3, 4, 5, ... are called whole numbers. The set of whole numbers can be represented by \textit{W}. Clearly, every natural number is a whole number but 0 is a whole number which is not a natural number.
    \begin{equation}
        W = \{0, 1, 2, 3, 4, 5, \ldots\}
    \end{equation}
    
    \subsection*{Integers}
    All counting numbers and their negatives including zero are known as integers. \textit{Z} or \textit{I}. The set of integers can be represented by
    \begin{equation}
        Z \text{ or } I = \{\ldots, -4, -3, -2, -1, 0, 1, 2, 3, 4, \ldots\}
    \end{equation}
    \begin{itemize}
        \item \textbf{Positive Integers}: The set \textit{I$^+$} = \{1, 2, 3, 4, \ldots\} is the set of all positive integers.
        \item \textbf{Negative Integers}: The set \textit{I$^-$} = \{-1, -2, -3, \ldots\} is the set of all negative integers. 0 is neither positive nor negative.
        \item \textbf{Non-negative Integers}: The set \{0, 1, 2, 3, \ldots\} is the set of all non-negative integers.
    \end{itemize}
    Note: Only whole numbers are integers. Therefore; fractions, decimals, percent and exponents can never be integers.
    
    \subsection*{Rational Numbers}
    The numbers of the form \textit{p/q}, where \textit{p} and \textit{q} are integers and \textit{q} $\neq$ 0, are known as rational numbers, e.g. 
    \begin{equation}
        \frac{4}{7}, \frac{3}{2}, \frac{5}{8}, \frac{1}{3}, \frac{2}{3}
    \end{equation}
    etc. The set of all rational numbers is denoted by \textit{Q}.
