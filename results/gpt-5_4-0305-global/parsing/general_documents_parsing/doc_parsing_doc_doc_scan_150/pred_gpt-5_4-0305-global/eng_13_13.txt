\bigskip

\noindent
\begin{minipage}[t]{0.48\textwidth}
\textbf{\underline{Mean}}

\[
\bar{x}=\frac{1}{n}\sum_{i=1}^{n}x
\]

\textbf{\underline{Standard Error of the Mean}}$^{*}$

\[
SE_{\bar{x}}=\frac{s}{\sqrt{n}}
\]

\textbf{\underline{Laws of Probability}}

If A and B are mutually exclusive, then:

\[
P(A \text{ or } B)=P(A)+P(B)
\]

If A and B are independent, then:

\[
P(A \text{ and } B)=P(A)\times P(B)
\]

\textbf{\underline{Hardy-Weinberg Equations}}

\[
p^2+2pq+q^2=1
\]

\[
p+q=1
\]

\end{minipage}
\hfill
\begin{minipage}[t]{0.48\textwidth}
\textbf{\underline{Standard Deviation}}$^{*}$

\[
S=\sqrt{\frac{\sum (x_i-\bar{x})^2}{n-1}}
\]

\textbf{\underline{Chi-Square}}

\[
X^2=\sum \frac{(o-e)^2}{e}
\]

Mode = value that occurs most frequently in a data set

\medskip

Median = middle value that separates the greater and lesser halves of a data set

\medskip

Mean = sum of all data points divided by number of data points

\medskip

Range = value obtained by subtracting the smallest observation (sample minimum) from the greatest (sample maximum)
