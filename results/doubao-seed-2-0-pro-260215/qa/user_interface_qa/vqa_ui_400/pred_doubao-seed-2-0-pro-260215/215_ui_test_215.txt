The three basic approaches modern search engines use to organize websites are:
1.  **Crawling**: Search engine crawlers follow links to browse the (currently over 60 trillion, growing) pages that make up the web.
2.  **Sorting**: Collected results are sorted by a variety of ranking factors.
3.  **Indexing**: The sorted pages are stored in an index, so search engines can match user queries to relevant results much faster.
