\section*{1. NUMBER SYSTEM}

\begin{center}
\textit{``NUMBER RULES THE UNIVERSE.'' PYTHAGORAS}
\end{center}

\subsection*{Natural Numbers}

These are also called \textit{Counting numbers} $1, 2, 3, 4, 5, \ldots$ and are known as natural numbers. The set of all natural numbers can be represented by $\mathbb{N}$.
\[\mathbb{N} = \{1, 2, 3, 4, 5, \ldots\}\]

\subsection*{Whole Numbers}

If we include 0 among the natural numbers, then the numbers $0, 1, 2, 3, 4, 5, \ldots$ are called whole numbers. The set of whole numbers can be represented by $\mathbb{W}$. Clearly, every natural number is a whole number but 0 is a whole number which is not a natural number.
\[\mathbb{W} = \{0, 1, 2, 3, 4, 5, \ldots\}\]

\subsection*{Integers}

All counting numbers and their negatives including zero are known as integers. $\mathbb{Z}$ or $\mathbb{I}$. The set of integers can be represented by
\[\mathbb{Z} \text{ or } \mathbb{I} = \{\ldots, -4, -3, -2, -1, 0, 1, 2, 3, 4, \ldots\}\]

\begin{itemize}
    \item \textbf{Positive Integers:} The set $\mathbb{I}^+ = \{1, 2, 3, 4, \ldots\}$ is the set of all positive integers.
    \item \textbf{Negative Integers:} The set $\mathbb{I}^- = \{-1, -2, -3, \ldots\}$ is the set of all negative integers. 0 is neither positive nor negative.
    \item \textbf{Non-negative Integers:} The set $\{0, 1, 2, 3, \ldots\}$ is the set of all non-negative integers.
\end{itemize}

\textbf{Note:} Only whole numbers are integers. Therefore, fractions, decimals, percent and exponents can never be integers.

\subsection*{Rational Numbers}

The numbers of the form $p/q$, where $p$ and $q$ are integers and $q \neq 0$, are known as rational numbers, e.g. $\dfrac{4}{7}, \dfrac{3}{2}, -\dfrac{5}{8}, \dfrac{0}{1}, -\dfrac{2}{3}$, etc. The set of all rational numbers is denoted by $\mathbb{Q}$.
