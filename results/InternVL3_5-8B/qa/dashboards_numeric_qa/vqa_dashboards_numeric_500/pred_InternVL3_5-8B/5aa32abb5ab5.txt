In the VMware vSphere vs. Microsoft Hyper-V comparison chart, the number of Failover nodes for Microsoft Hyper-V Standard is 32, for Microsoft Hyper-V Enterprise it is 32, for Microsoft Hyper-V Datacenter it is 16, and for Microsoft Hyper-V with Microsoft System Center it is 16. Therefore, two versions of Microsoft Hyper-V have Failover nodes equal to 16: Microsoft Hyper-V Datacenter and Microsoft Hyper-V with Microsoft System Center.
