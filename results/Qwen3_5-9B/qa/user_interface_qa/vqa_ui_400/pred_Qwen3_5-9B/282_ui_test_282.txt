The **Swab test** (also called the **Laboratory test** or **Home test**, depending on where it’s done) is the one that indicates whether a person needs to be isolated or can return to work.

According to the infographic:

> **What does it mean?**  
> Indicates if a person:  
> • needs to continue to self isolate  
> • can return to work  
> • needs to be re-tested

This section is under the **Swab test** column (black background), which answers the question: *“Do I have it?”*

---

✅ So, the correct answer is: **Swab test**

*(Note: The antibody test — “Have I had it?” — only tells you if you’ve been infected in the past, not whether you’re currently contagious or need isolation.)*
