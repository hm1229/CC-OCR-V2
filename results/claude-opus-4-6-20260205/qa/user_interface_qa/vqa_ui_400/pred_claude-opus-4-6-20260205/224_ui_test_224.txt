# Search Technologies Before 1995

Based on the infographic's "A Quick History of Search Engine Technology" section, the search technologies that existed before 1995 were:

1. **Archie (Pre-WWW)** - Starting around **1990**, it was the first search engine that didn't really crawl sites, but it kept an index of public FTP servers so users could find and download files.

2. **Lycos** - Appearing around **1994**, the Lycos index grew from 54,000 documents in July to 60 million by 1996.

These two search technologies were established before 1995, with **Yahoo!** and **Google** coming later (Yahoo! in **1996** and Google in **1998**).
