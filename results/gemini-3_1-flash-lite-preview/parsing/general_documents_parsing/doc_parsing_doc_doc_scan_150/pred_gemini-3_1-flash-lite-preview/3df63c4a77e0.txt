### Physics Formula Sheet

#### Mechanics

*   $x = x_0 + v_{x0}t + \frac{1}{2}a_xt^2$
*   $v = v_0 + at$
*   $v^2 - v_0^2 = 2a(x - x_0)$
*   $\vec{a} = \frac{\sum \vec{F}}{m} = \frac{\vec{F}_{net}}{m}$
*   $|\vec{F}_{friction}| \leq \mu |\vec{F}_{Normal}|$
*   $\vec{p} = m\vec{v}$
*   $\Delta \vec{p} = \vec{F}\Delta t$
*   $KE = \frac{1}{2}mv^2$
*   $\Delta PE = mg\Delta y$
*   $\Delta E = W = Fd\cos\theta$
*   $a_c = \frac{v^2}{r}$
*   $\theta = \theta_0 + \omega_0t + \frac{1}{2}\alpha t^2$
*   $\omega = \omega_0 + \alpha t$
*   $T = \frac{2\pi}{\omega} = \frac{1}{f}$
*   $v = f\lambda$
*   $x = A\cos(2\pi ft)$
*   $\vec{\alpha} = \frac{\sum \vec{\tau}}{I} = \frac{\vec{\tau}_{net}}{I}$
*   $\vec{\tau} = r \times F$
*   $L = I\omega$
*   $\Delta L = \tau\Delta t$
*   $|\vec{F}_{spring}| = k|\vec{x}|$
*   $PE_{spring} = \frac{1}{2}kx^2$
*   $T_{spring} = 2\pi\sqrt{\frac{m}{k}}$
*   $T_{pendulum} = 2\pi\sqrt{\frac{\ell}{g}}$
*   $|\vec{F}_{gravity}| = G\frac{m_1m_2}{r^2}$
*   $|\vec{F}_{gravity}| = m\vec{g}$
*   $PE_{gravity} = -G\frac{m_1m_2}{r}$
*   $\rho = \frac{m}{V}$
*   $KE = \frac{1}{2}I\omega^2$

#### Electricity

*   $|\vec{F}_E| = k\frac{q_1q_2}{r^2}$
*   $I = \frac{\Delta q}{\Delta t}$
*   $R_{series} = R_1 + R_2 + \dots + R_n$
*   $\Delta V = IR$
*   $P = I\Delta V$
*   $R = \frac{\rho\ell}{A}$
*   $\frac{1}{R_{parallel}} = \frac{1}{R_1} + \frac{1}{R_2} + \dots + \frac{1}{R_n}$

#### Geometry

*   **Rectangle:** $A = bh$
*   **Rectangular Solid:** $V = \ell wh$
*   **Triangle:** $A = \frac{1}{2}bh$
*   **Circle:** $A = \pi r^2$, $C = 2\pi r$
*   **Cylinder:** $V = \pi r^2\ell$, $S = 2\pi r\ell + 2\pi r^2$
*   **Sphere:** $V = \frac{4}{3}\pi r^3$, $S = 4\pi r^2$

#### Trigonometry

*   $c^2 = a^2 + b^2$
*   $\sin\theta = \frac{a}{c}$, $\cos\theta = \frac{b}{c}$, $\tan\theta = \frac{a}{b}$
