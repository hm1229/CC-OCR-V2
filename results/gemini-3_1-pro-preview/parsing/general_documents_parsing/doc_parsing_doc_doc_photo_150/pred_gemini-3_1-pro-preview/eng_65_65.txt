\textbf{\Large 1. NUMBER SYSTEM}

\textit{"NUMBER RULES THE UNIVERSE." PYTHAGORAS}

\textbf{Natural Numbers}
\begin{itemize}
    \item[] These are also called \textit{Counting numbers} $1, 2, 3, 4, 5, \dots$ and are known as natural numbers.
    \item[] The set of all natural numbers can be represented by $N$.
    \item[] \centering $N = \{1, 2, 3, 4, 5, \dots\}$
\end{itemize}

\textbf{Whole Numbers}
\begin{itemize}
    \item[] If we include $0$ among the natural numbers, then the numbers $0, 1, 2, 3, 4, 5, \dots$ are called whole numbers. The set of whole numbers can be represented by $w$. Clearly, every natural number is a whole number but $0$ is a whole number which is not a natural number.
    \item[] \centering $W = \{0, 1, 2, 3, 4, 5, \dots\}$
\end{itemize}

\textbf{Integers}
\begin{itemize}
    \item[] All counting numbers and their negatives including zero are known as integers. $Z$ or $I$.
    \item[] The set of integers can be represented by
    \item[] \centering $Z \text{ or } I = \{\dots -4, -3, -2, -1, 0, 1, 2, 3, 4, \dots\}$
    \item[$\bullet$] \textbf{Positive Integers:} The set $I+ = \{1, 2, 3, 4, \dots\}$ is the set of all positive integers.
    \item[$\bullet$] \textbf{Negative Integers:} The set $I- = \{-1, -2, -3, \dots\}$ is the set of all negative integers. $0$ is neither positive nor negative.
    \item[$\bullet$] \textbf{Non-negative Integers:} The set $\{0, 1, 2, 3, \dots\}$ is the set of all non-negative integers.
\end{itemize}
\textbf{Note:} Only whole numbers are integers. Therefore,: fractions, decimals, percent and exponents can never be integers.

\textbf{Rational Numbers}
\begin{itemize}
    \item[] The numbers of the form $p/q$, where $p$ and $q$ are integers and $q \neq 0$, are known as rational numbers, e.g. $\frac{4}{7}, \frac{3}{2}, -\frac{5}{8}, \frac{0}{1}, -\frac{2}{3}$, etc. The set of all rational numbers is denoted by $Q$.
\end{itemize}
